# 子组件更新父组件传参数据，DOM不更新解决小技巧（数组类型数据）

WARNING

该方法适用于vue2

# vue父组件向子组件传参，子组件更改该数据，DOM不更新，解决小技巧(一般传参为数组会有DOM不更新情况)

// fatherData数据如下

fatherData: [
  {
    icon: {
      type: `${iconType}-home`,
    },
    name: '主页',
    iconImg: {
      // default: this.imgUrl + 'ichengdu/home.png',
      // selected: this.imgUrl + 'ichengdu/home-fill.png'
    }
  },
  {
    icon: {
      type: `${iconType}-briefcase`,
    },
    name: '商城',
    iconImg: {
      // default: this.imgUrl + 'ichengdu/shop.png',
      // selected: this.imgUrl + 'ichengdu/shop-fill.png'
    }
  },
  {
    icon: {
      type: `${iconType}-contact`,
    },
    name: '我的',
    iconImg: {
      default: '',
      selected: ''
    }
  }
]

// 子组件接受的参数

prop: [
  'fatherData'
]

-子组件的data内新建一个变量data，如下

// 子组件data
data () {
  return {
    data: []
  }
}

-在子组件created该生命周期阶段，用data接受父组件传来的并修改的数据

// 处理父组件传来的参数

this.fatherData.forEach(
  (item, i) => {
    item.icon.size = this.iconSize
    item.flag = false
    if (i === 0) {
      item.flag = true
    }
    this.data.push(item)
  }
)

-需要改变flag值并让DOM更新

// change函数接受索引并改变该索引对应的flag值，并更新DOM
// 先清空data内部数据
// 再赋值给data，即可触发DOM更新

change (i) {
  this.data = []
  this.tabBarData.forEach(
    (_item, _i) => {
      _item.flag = false
      if (i === _i) {
        _item.flag = true
      }
      this.data.push(_item)
    }
  )
}
