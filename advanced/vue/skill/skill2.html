  • 关于vue的小技巧

    • 子组件更新父组件传参数据，DOM不更新解决小技巧（数组类型数据）
    • 父组件无法向动态组件直接通过this.$refs获取子组件内部的节点的解决方法
      • 只需要向动态组件添加一个ref="..."即可解决

# 父组件无法向动态组件直接通过this.$refs获取子组件内部的节点的解决方法

WARNING

该方法适用于vue2

# 只需要向动态组件添加一个ref="..."即可解决

// 这是父组件中动态组件

<component :is="tabsContent" ref="tabsContent" />

// 这是子组件A

<template>
  <div>
    <div ref="A">
    </div>
  </div>
</template>

// 这是子组件B

<template>
  <div>
    <div ref="B">
    </div>
  </div>
</template>

// 父组件通过以下方法即可获取动态组件下的ref
// name = 'A' | 'B'

this.$refs.tabsContent.$refs[name]

← 子组件更新父组件传参数据，DOM不更新解决小技巧（数组类型数据）
